# COOP4490-Huawei-AI-Applications-Project
This is a term project for a course offered by Huawei in Bahçeşehir University.
The project purpose is to create an image classifier using 4 model different architectures  and evaluate the results using the cross valuation method with 5 folds.
This project got 100/100.
